# Pick Value Lab — Marvel Rivals (Season 9)

Interactive draft simulator for *Marvel Rivals*: estimates **adjusted hero pick value** from
ranked matches, then lets you build both team drafts and see the model's expected result.

**Live:** https://plivdan.github.io/pick-value-lab/

A single self-contained HTML page (all data and images embedded). The hero, team-up, and
role-structure values come from a ridge-logistic hero-impact model fit on 121,830 Season 9
ranked matches (Diamond through One Above All, Grandmaster-heavy), controlling for both teams'
heroes, team-up eligibility, role structure, map, mode, and repeated-player effects. Every value
carries a 95% bootstrap interval, shown as a whisker in the roster. Draft slots use the official
hero card art. Regenerated by `scripts/build_pick_value_lab.py` in the rivals-lab pipeline as new
data is crawled.
